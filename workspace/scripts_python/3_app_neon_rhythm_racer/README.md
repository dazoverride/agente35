# Neon Rhythm Racer

Un juego de ritmo y reacción desarrollado en Python con Pygame.

## Descripción
En **Neon Rhythm Racer**, tu objetivo es esquivar notas mal colocadas y acertar las que caen en la posición correcta presionando la tecla correspondiente.

### Controles
- **A, S, D, F**: Mueve tu nave hacia el carril correspondiente.
- **Espacio**: Iniciar juego o Reiniciar tras Game Over.
- **Esc**: Salir.

## Cómo Jugar
1. Presiona **Espacio** en el menú principal.
2. Observa las notas de colores que caen desde arriba.
3. Presiona la tecla asociada al color de la nota cuando esté justo frente a tu nave.
4. Acertar suma puntos; fallar reduce tus vidas.
5. Alcanza 2000 puntos o agota tus vidas para terminar.

## Requisitos
- Python 3.x
- Librería `pygame` (Instalar con: `pip install pygame`)
