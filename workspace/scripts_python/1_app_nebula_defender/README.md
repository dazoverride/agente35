# Nebula Defender

Un juego de defensa de torreta estilo arcade desarrollado en Python con `pygame`.

## Cómo jugar
1. **Objetivo**: Protege el núcleo central de la nebulosa (círculo cian) de los asteroides grises.
2. **Controles**:
   - `Flecha Izquierda` / `Flecha Derecha`: Rotar el láser para apuntar.
   - Disparar automáticamente cuando el láser apunta al asteroide.
3. **Reiniciar**: Presiona la tecla `R` al finalizar el juego.

## Características
- Generación procedural de asteroides con dificultad progresiva.
- Sistema de partículas para explosiones.
- Fondo de estrellas estáticas.
- Código modular y comentado siguiendo las normas del Arquitecto de Software.
