# Visualizador de Red Neurale Aleatoria

## Descripción
Este script genera una topología de red neuronal aleatoria, calcula métricas de centralidad y genera una visualización gráfica en PDF para análisis estructural.

## Requerimientos
- `matplotlib`
- `networkx`

## Uso
Ejecuta el script directamente:

```bash
python 24_visualizador_red_neural.py
```

## Salida
- `red_neural.json`: Contiene la representación gráfica (JSON node-link) y métricas calculadas.
- `visualizacion_red.pdf`: Gráfico visual de la red con nodos coloreados por tipo y aristas ponderadas.
