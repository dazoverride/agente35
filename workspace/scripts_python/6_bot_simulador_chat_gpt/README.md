# Bot Simulador Chat GPT

Un simulador básico de chat que imita la interacción de un bot inteligente.

## Características
- Respuestas dinámicas basadas en el contexto del mensaje.
- Reconocimiento de saludos, preguntas y comentarios.
- Interfaz de línea de comandos (CLI) interactiva.

## Cómo ejecutarlo
1. Asegúrate de tener Python 3.x instalado.
2. Ejecuta el script con el siguiente comando:
   ```bash
   python 6_bot_simulador_chat_gpt/main.py
   ```
3. Inicia la conversación escribiendo mensajes.

## Requisitos
- Python 3.x
- Sin dependencias externas.
