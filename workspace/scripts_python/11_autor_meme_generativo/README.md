# Generador de Memes CLI (ID: 11)

## Descripción
Herramienta de línea de comandos que genera archivos de texto con estructuras de memes, combinando palabras al azar para crear contenido humorístico o absurdo.

## Dependencias
- Python 3.x (sin librerías externas)

## Uso
1. Clona o ejecuta el script `11_autor_meme_generativo/main.py`.
2. El script creará automáticamente una carpeta `memes_generados`.
3. Dentro encontrarás 5 archivos `.txt` con el formato de meme.

## Ejemplo de salida
```
Top Text: banana
Bottom Text: pan
```

## Notas
Este script es una herramienta creativa para generar ideas o contenido de prueba sin necesidad de imágenes reales.
