# Explorador Espacial: Busca Tesoros

Un juego de simulación espacial desarrollado con **Pygame**.

## Objetivo
Explora el espacio, recolecta planetas para recargar tu combustible y evita chocar con las naves enemigas rojas.

## Controles
- **WASD** o **Flechas**: Mover la nave.
- **Espacio**: Disparar láseres (consume combustible).
- **R**: Reiniciar el juego si te quedas sin combustible.

## Mecánicas
- **Combustible**: Se agota con el tiempo y al disparar. Los planetas verdes/azules/aurangec recargan tu tanque al tocarlos.
- **Enemigos**: Las naves rojas te hacen perder combustible al chocar.
- **Puntuación**: Ganas puntos al destruir enemigos y al tocar planetas.

## Requisitos
- Python 3.x
- Librería `pygame` (`pip install pygame`)

## Cómo Jugar
1. Clona o descarga los archivos.
2. Ejecuta `python main.py`.
3. ¡Diviértete explorando el vacío!
